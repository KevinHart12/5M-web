Regeln für FiveM RP Server

Hier sind die grundlegenden RP Regeln die du wissen solltest um auf RP Servern zu Spielen

RDM: Random Deathmatch
Random Deathmatch,Zivilisten/Personen ohne Triftigen grund erschießen.z.b eine Person läuft auf der Straße und sie wird ohne Schussankündigung erschossen dann zählt das als Random Deathmatch>
Schussankündigung:Schusscall,Call
Schusscall,bevor man eine Person erschießt gibt man ihr einen (call) z.B. Hände hoch oder es knallt geht sie nicht die Forderungen nach so kann sie nach erschossen werden. Es herscht aber eine geraume zeit wo man der Person zeit lassen muss um zu agieren.(15sek) Ein Call hält 10min lang wenn dieser erfällt ist der gegebene Call nicht mehr gültig.
VDM:Vehicle Deathmatch
Vehicle Deathmatch ist wenn man jemanden wie beim RDM ohne Hintergrund mit einem Fahrzeug umfährt.Schützt es IC:In Charakter dein Leben dann darfst jemanden mit nicht so hoher geschwinigkeit umfahren.Lässt sich jemand von dir umfahren dann ist es nicht deine schuld
FailRP
Eine Situation spielen die im echten leben nicht funktionieren würde Bsp.:bei eine Verfolgungsjagt einen Berg hochfahren landen und normal weiterfahren als wäre nix passiert.Sowas nennt man auch GTA Driving.
Wörter wie OOC,IC,Roboter,Kopfschmerzen,etc.
Es gibt wörter im RP die man im RP sagen um z.b. zu sagen es laggt:ich habe kopfschmerzen,OOC: out of charkter,IC: In Charakter,Roboter: Mikrofon laggt,Staat:Server,Universum:FiveM,Regelwerk:Bibel,Gesetzbuch,Europa:Deutschland
Eigener Charakter
Man kann im RP einen eigenen Charakter erfinden:eigener Name,eigenes Alter,eigene Geschichte:z.B.Ich heiße Levin Benz und ich bin hier auf den Staat gekommen weil man in meinem ALten Staat(Server) nicht gefallen hat ich möchte neue leute kennenlernen etc, oft haben Server eine "Einreise" wo man ein Gespräch macht warum man auf den Staat will was ist deine Geschichte,danach folgt eine Regel abfragung.Es gibt aber auch einige server die eine Allowlist auf dem Discordserver haben dort kannst du normal reden es herscht kein RP in einem Discordallowlist gespräch.
Server und ihre Regeln
Jeder Server hat seine eigenen Regeln deswegen sollte man sich im Server Forum oder auf dem Discordserver deren eigenen Regeln nochmal durchlesen.
